About ImageDataExtractor

ImageDataExtractor is a tool for automatically extracting quantitative data from microscopy images.

Give it an image, and it will identify regions of interest and calculate particle size metrics and the Radial Distribution Function (RDF).

Figure mining

ImageDataExtractor harnesses the power of ChemDataExtractor to automatically detect relevant images from a given document.

Machine-learning methods such as conditional random fields are used in combination with custom dictionaries and rule-based parsing grammars to reduce figure captions to a machine-readable form.

The result is then parsed using terms given by the user and all matches downloaded.

Particle identification

Individual particles are isloated using our custom contour detection algorithm and converted to real values by scale-bar intepretation

Compound images are first split into their constituent images, and Optical Character Recognition performed on each scalebar to obtain scale information

Particles are found using a threshold to find boundaries and fitting ellipses to the resultant outline.

Metric calculation

Average particle size and RDF are calculated.

Open source

ImageDataExtractor is available as an open source python package that you can download and use for free.

Check out the documentation for help getting started.
